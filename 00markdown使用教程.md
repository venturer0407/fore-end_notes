# markdown 使用教程
## 标题语法
 #一级标题    ##二级标题
 ###三级标题    ####四级标题
 #####五级标题    ######六级标题
**正文直接打出来就可以了**
## 字体语法
斜体ctrl+i *斜体* 
加粗ctrl+b **加粗**
